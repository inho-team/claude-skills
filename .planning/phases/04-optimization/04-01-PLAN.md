<task type="auto">
  <name>Task 1: Implement ContextMemo Utility in state.mjs</name>
  <files>hooks/scripts/lib/state.mjs</files>
  <action>
    Add the following utility functions to `hooks/scripts/lib/state.mjs` to manage the `memo` field in `unified-state.json`:
    - `getMemo(state, filePath)`: Returns cached content for a path.
    - `setMemo(state, filePath, content)`: Stores content in `state.memo[filePath]`.
      - **Enforce 10KB limit per file**. If larger, do not store.
      - **Enforce 100KB total limit** for the `memo` object. Use FIFO or LRU if necessary to stay within budget.
    - `clearMemo(state, filePath)`: Deletes entry from `state.memo`.

    Ensure these functions operate on the `state` object passed as an argument (caller handles persistence).
  </action>
  <verify>
    <automated>node -e 'import("./hooks/scripts/lib/state.mjs").then(m => { let s = {}; m.setMemo(s, "test.txt", "a".repeat(100)); if (m.getMemo(s, "test.txt") === "a".repeat(100)) console.log("SUCCESS") })'</automated>
  </verify>
  <done>Utility functions exported and functional with size limits.</done>
</task>

<task type="auto">
  <name>Task 2: Integrate ContextMemo into Tool Hooks</name>
  <files>hooks/scripts/pre-tool-use.mjs, hooks/scripts/post-tool-use.mjs</files>
  <action>
    Modify the hooks to automate memoization:
    - **hooks/scripts/post-tool-use.mjs**:
      - On successful `Read` tool: If content length <= 10KB, call `setMemo` for the path.
      - On successful `Write` or `Edit` tool: Call `clearMemo` for the path to invalidate stale cache.
    - **hooks/scripts/pre-tool-use.mjs**:
      - If tool is `Read`:
        - Check `getMemo` for the requested file path.
        - If present, inject a hint: `[MEMO] Content for ${filePath} is already available in the ContextMemo. Prefer using it instead of reading from disk if possible.`
        - For critical files (`CLAUDE.md`, `package.json`, `TASK_REQUEST_*.md`), make the hint more prominent.

    Ensure these hooks correctly read and write the updated `state` to `unified-state.json`.
  </action>
  <verify>
