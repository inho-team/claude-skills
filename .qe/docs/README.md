# .qe/docs — Domain Knowledge

이 디렉터리는 프로젝트 도메인 지식 문서를 담습니다. session-start hook이 작업과 관련된 도메인 문서를 여기서 찾도록 설계됐습니다.

## 용도

- 프로젝트 비즈니스 용어집
- 외부 API/프로토콜 스펙 요약
- 팀 내부 운영 절차/컨벤션 메모
- 재현 가능한 실험/벤치마크 설계 노트

## 현재 상태

_아직 등록된 문서 없음._ 새 문서는 짧은 파일명(예: `domain-glossary.md`, `api-conventions.md`)과 `# 헤더` + `## 개요` 구조로 추가하세요.
